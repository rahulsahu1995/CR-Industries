Project Name: C R INDUSTRIES Website

Tech Stack: React (Vite), Tailwind CSS, Framer Motion (for animations), Lucide React (for icons), and SendGrid API.

Instruction: Please build a modern, responsive, and high-performance website for "C R INDUSTRIES" based on the following technical requirements:


Design & Color Palette:
Primary Colors: Gray, #0096C7, and #03045E.
Theme Logic: Use #03045E sparingly in Light Mode.
Dark Mode: Implement a fully functional Light/Dark mode toggle.


Functionality & Email Integration:
No Database: Do not use any database (MongoDB, Firebase, etc.).
Email System: Use SendGrid API to handle form submissions. Create a Contact Form that sends user details directly to a specified email via SendGrid.


Navigation & Scroll Behavior:
Auto-Scroll to Top: Implement a global "Scroll to Top" behavior on all routes. Whenever a route changes (or the same link is clicked again), the page must smoothly scroll to the top automatically.
Custom Scrollbars: Hide all scrollbars globally across the entire website for a clean, app-like feel.


Responsive Design & Animations:
The website must be fully responsive across all devices (Mobile, Tablet, Desktop).
Mobile Experience: On mobile devices, prioritize auto-animations (entrance animations) using Framer Motion. Hover effects can be disabled or minimized for mobile touch targets.


Security & Environment Variables:
All sensitive keys, including SendGrid API keys and Template IDs, must be stored in a .env file.
Ensure the code references these variables using the correct environment naming convention (e.g., VITE_SENDGRID_API_KEY).


UI Components:
Include a Navbar with the company name "C R INDUSTRIES", a Hero section, Services, About Us, and a functional Contact Form.
Ensure smooth transitions and a premium "industrial-modern" aesthetic.
